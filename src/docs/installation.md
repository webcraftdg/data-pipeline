# Installation

---

## 🎯 Objectif

Installer le composant `webcraftdg/data-pipeline` dans un projet PHP.

---

## 📦 Prérequis

- PHP 8.3 ou supérieur
- Composer installé

---

## 📥 Installation via Composer

Dans votre projet, exécutez :

```bash
composer require webcraftdg/data-pipeline:^1.0.0
```
